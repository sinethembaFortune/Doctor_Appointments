## Installation

1. Clone the repository: `git clone https://github.com/sinethembaFortune/doctor-appointment-website.git`
2. Navigate to the project directory: `cd doctor-appointment-website`
3. Open the `index.html` file in your web browser.

## Usage

2. Log in with Doctor credentials.
3. Schedule a doctor appointment by providing necessary details.
4. View and manage appointments.
